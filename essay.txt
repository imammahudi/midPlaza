PROBLEM SOLVING
Dalam memberikan solusi ketika di waktu yang sama terjadi request yang dapat mengakibatkan overload maka di case ini ada beberapa bagian :
1. Frontend 
 - Mengimplementasikan HTTP Caching
2. Backend
 - Memastikan bahwa Requirements Server telah memadai
 - Membuat queue
 - Implement Rate Limiting
